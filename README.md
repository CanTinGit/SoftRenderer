# SoftRenderer(软渲染器)

## 运行要求
运行此项目，请确认您的电脑已经正确安装并配置了opencv（用于读取贴图）

## 具体功能
- 加载obj格式模型
- 贴图
- 漫反射，环境光，高光
- 阴影
- 背面剔除

---

## Requirement
If you want to run this program, please make sure you have already installed opencv and set it up

## Functions
- Load models from .obj
- Texture
- Diffuse light, ambient light, specular light
- Shadow map
- BackCulling
